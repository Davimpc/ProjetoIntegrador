Botão de Emergência - App de Locomoção Urbana
Este projeto é uma funcionalidade desenvolvida para um aplicativo de locomoção urbana para pedestres.
Garante maior segurança para o aplicativo, pois permite ao usuário acionar rapidamente um botão de emergência e ligar para contatos importantes como polícia, bombeiros ou família.
Feito em React Native.

Telas do projeto: 
MapaEmergenciaTela: Tela com mapa e o botão de emergência disponível no canto inferior direito.
ConfiguracoesTela: Tela de configurações para definir o comportamento do botão e cadastrar contatos.
EmergenciaVisualTela: Tela com o botão de emergência ativado, com contatos listados para ligação imediata.
